# B@rkl4ys IB — Institutional Client Portal (v2.2 Final)

A sophisticated institutional client portal with comprehensive portfolio management, market research, and admin capabilities.

## Features
- **Dashboard**: YTD/3M/6M performance tracking with Recent Market Activity
- **Investments**: Multi-asset portfolio (Stocks, Bonds, Funds, CDs)
- **Research**: Live market indices with UK/US/Global filtering
- **Documents**: AML, Contracts, and document management
- **Deposit**: Bank transfer instructions with QR code
- **Support**: Integrated ticket system
- **Settings**: Profile, 2FA, session monitoring
- **Back-Office**: Admin analytics dashboard

## Tech Stack
React 18, TypeScript, Tailwind CSS, shadcn/ui, Recharts

## Getting Started
```bash
npm install
npm run dev
```

## Disclaimer
This portal is a demonstration environment. All data and entities are fictitious.
